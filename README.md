TinyPWMTone
===========

Port of PiPWMTone to the ATtiny13A micro controller


Wiring
------

```
          GND  GND
           |    |
VCC       BTN  SPK
 |    |    |    |
-----------------
|               |
|   ATtiny13A   |
|               |
-----------------
 |    |    |   |
    LED2 LED1 GND
      |    |
     GND  GND
```
